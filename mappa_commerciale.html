AI e Machine Learning
8 moduli AI e 8 moduli ML per personalizzazione e ottimizzazione
💬
Message Personalizer
Genera messaggi B2B personalizzati via OpenAI. Retry con temperatura crescente, semantic similarity check per evitare ripetizioni, max chars enforced.
Supporto multilingua: IT, EN, FR, ES, NL
Template fallback se AI non disponibile
Semantic checker per deduplica
✉
Invite Note Personalizer
Note invito personalizzate con A/B testing multi-armed bandit (Thompson sampling). 8 template IT, 5 EN, 3 FR con varianti AI.
A/B bandit per selezione variante
Segmentazione lead automatica
Timing optimizer per orario invio
🎯
Lead Scorer
Scoring automatico 1-100 basato su ruolo, seniority e azienda target. Due score: confidenceScore (lavora davvero li?) e leadScore (quanto vale?).
Criteri personalizzabili per lista
Penalizzazione ex-dipendenti
Output JSON strutturato con reason
📈
Timing Optimizer
ML model per ottimizzare l'orario di invio messaggi e inviti. Considera timezone del lead, storico risposte, giorno della settimana.
Timezone detection automatica
Attribution tracking per conversioni
Adaptive scheduling basato su dati
🖱
Mouse Generator
Generazione traiettorie mouse con curve Bezier realistiche. Ogni movimento ha velocita, accelerazione e micro-deviazioni umane.
Curve Bezier con control points casuali
Offset click dal centro elemento
Stato mouse persistente per pagina (WeakMap)
👁
Vision Solver (Captcha)
Computer vision via Ollama (LLaVA) o OpenAI per risolvere captcha e navigare UI non accessibili via selettori DOM standard.
Object coordinate detection
Viewport bounds clamping
Fallback factory (Ollama/OpenAI)
Sicurezza e anti-detection
Sistema multi-livello per protezione account e simulazione comportamento umano
📈
Risk Engine (score 0-100)
Scoring in tempo reale basato su: error rate (peso 40), selector failure rate (peso 20), pending ratio (peso 25), challenge count (peso 10), invite velocity (peso 15). Azioni: NORMAL, LOW_ACTIVITY, WARN, STOP.
🔒
Fingerprint Pool
Pool di fingerprint desktop e mobile con selezione deterministica per account. Coerenza TLS: filtro automatico UA/browser engine per evitare mismatch rilevabili.
👨
Human Behavior Simulator
Delay log-normale, typing con typo, reading scroll, decoy actions (random like, profile visit), coffee break, wind-down a fine sessione, missclick accidentali.
🕒
Strategy Planner settimanale
Profili attivita per giorno: lunedi alto inviti, giovedi focus messaggi, weekend zero. Multiplier per ogni giorno su inviteFactor e messageFactor.
🚀
HTTP Throttler adattivo
Monitora response time di LinkedIn in tempo reale. Se rallenta: prima slow (extra delay), poi 3 slow consecutivi = abort sessione. Smart batch sizing con riduzione -30%.
🚨
Incident Manager
Gestione automatica di challenge, quarantena account, pausa automazione con timer, proxy failure escalation (3 failure = stop 7 giorni + alert Telegram critico).
